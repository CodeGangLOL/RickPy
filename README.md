## This repository requires:
> pygame (>2.6.4)  
> Python (python3 3.9+)

## How to Build Yourself:

1. <i>Installation</i>  
> run `pip install -r requirements.txt`  
> ---or---  
> run `install.bat` (Only on Windows)  
> ---or---  
> run `chmod install.sh && ./install.sh` (Linux BASH)
> ---or---
> run `install.sh` via Code Editor

2. <i>Building *.pyc Process</i>
> run the VS Code Task  
> Build Manually
> |-> Option 1:
> |     |-> Build via Python3 compileall cmd (`python3 -m py_compile script.py script.pyc`)
